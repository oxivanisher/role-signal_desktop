signal_desktop
==============

This role installs signal desktop.

Example Playbook
----------------
```yaml
- name: Install signal desktop
  hosts: client
  collections:
    - oxivanisher.linux_desktop
  roles:
    - role: oxivanisher.linux_desktop.signal_desktop
```
